Game Paths and Endings

  • Run away: forest → run → Ending 1
  • Swim to shore: river → bridge → shore → Ending 2
  • Let the current take him: river → current → Ending 3
  • Fight: forest → sound → fight → Ending 4 or 7, depending on further choices:
    • Rest in a cave → Ending 6
    • Seek help → Ending 7
    • Keep moving despite injuries → Ending 4
  • Climb a tree: forest → sound → tree → Ending 5
  • Rest in a cave: fight → cave → Ending 6
  • Seek help: fight → rescue → Ending 7
  • Follow the scent: river → shallow → family → Ending 8 (Happy Ending)
